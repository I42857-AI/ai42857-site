## 心跳检查项

1. 有进行中的长任务？
2. 是否需要向用户汇报进度？
3. 今日 memory 文件是否需要更新？
4. 有新的大模型发布吗？（每天查一次，有好消息告诉用户）
5. 更新身份文件？→ 检查是否已更新，已更新静默跳过，未更新执行修改
6. Symlink 健康检查？→ 检查 ~/.agents/ 下所有 Symlink 目标是否存在
7. 虫洞检查？→ 确认沙箱 skills/ 与 output/ Symlink 指向母本有效

**如无需处理，回复 `HEARTBEAT_OK`**
